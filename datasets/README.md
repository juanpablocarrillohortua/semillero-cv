# Datasets

## Datasets Publicos Recomendados

| Dataset | Descripcion | Formato | URL |
|---------|-------------|---------|-----|
| COCO 2017 | Deteccion, segmentacion, keypoints | COCO JSON | https://cocodataset.org |
| SoccerNet | Futbol: deteccion, tracking, re-ID | Custom | https://www.soccer-net.org |
| SportsMOT | Multi-object tracking en deportes | MOT | https://github.com/MCG-NJU/SportsMOT |
| MPII Human Pose | Pose estimation | Custom | http://human-pose.mpi-inf.mpg.de |

## Datos del Semillero

Los datos del semillero NO se guardan en git. Usar directorios locales:

```
data/           # Datos de trabajo (gitignored)
outputs/        # Resultados de experimentos (gitignored)
```

Para datasets grandes, coordinar con el profesor para distribucion local en el lab.
